# Site Assembleia Doravante

Site institucional desenvolvido para modernizar a comunicação da comunidade religiosa.

** Site Online:** https://www.assembleiadoravante.com.br

##  Resultados na Comunidade
- Aprovação em pesquisa com membros
- Participantes na validação do projeto  
- Comunicação 24/7 para todas as gerações

## Stack
- Django + PythonAnywhere
- HTML/CSS/JavaScript
- Domínio personalizado + SSL
